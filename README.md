# 🚀 IntervAI – AI Powered Job Preparation Platform

**IntervAI** is a **Full Stack Generative AI web application** that helps users prepare for job interviews using AI-driven insights.

The platform allows users to **upload resumes, analyze job descriptions, detect skill gaps, generate interview questions, and create ATS-optimized resumes** using modern AI technologies.

Built with **React, Node.js, JWT Authentication, Gemini AI, MongoDB, and Puppeteer**, this project demonstrates how **Generative AI can be integrated into real-world career preparation tools.**

---

# 🌟 Features

* 🔐 Secure User Authentication using JWT
* 📄 Resume Upload & Processing
* 🤖 AI Resume Analysis
* 📊 Skill Gap Detection from Job Descriptions
* 💡 AI Generated Interview Questions
* 🧠 AI Interview Report Generation
* 📑 ATS Optimized Resume Builder
* 📥 Resume PDF Generation using Puppeteer

# 🧠 How IntervAI Works

1️⃣ Users register and log in securely using **JWT authentication**.

2️⃣ Users upload their **resume** and optionally provide a **job description**.

3️⃣ The system processes the resume using **AI-powered parsing and skill extraction**.

4️⃣ AI compares resume skills with job requirements and identifies **skill gaps**.

5️⃣ The platform generates **AI-powered interview questions and interview reports**.

6️⃣ Users can generate an **ATS-optimized resume in PDF format**.

# ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/FUNCTIONCALL-AGRIM/IntervAI.git
cd IntervAI
```

---

### 2️⃣ Install Dependencies

Frontend

```bash
cd Frontend
npm install
```

Backend

```bash
cd Backend
npm install
```

---

### 3️⃣ Environment Variables

Create a `.env` file inside the **Backend folder**

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
GEMINI_API_KEY=your_gemini_api_key
```

---

### 4️⃣ Run the Application

Start Backend

```bash
npm run dev
```

Start Frontend

```bash
npm run dev
```
